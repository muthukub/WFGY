# F1 Grounding Reproduction Pack v1

## Community reproduction bundle

This is a compact community reproduction pack for a simple F1 grounding case.

It is designed to make one small atlas-guided troubleshooting pattern easier to inspect and reuse.

---

## What this pack contains

This pack is expected to bundle together assets such as:

- a short case description
- one input fixture
- one expected output fixture
- one prompt example
- one short workflow recipe

This pack is intentionally small.

---

## What this pack demonstrates

This pack demonstrates one core idea:

> route the case to F1 first  
> then apply a grounding-first repair move

It does not claim to solve every retrieval or reasoning problem.

---

A clean minimal version of this pack can include:

## Expected result

Someone using this pack should be able to understand:

- what kind of case this is
- why it routes to F1
- what the first repair move is
- what a better output should look like

That is enough for a useful first reproduction pack.
